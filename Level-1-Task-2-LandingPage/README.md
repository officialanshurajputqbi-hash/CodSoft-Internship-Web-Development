📌 About
A fully responsive cybersecurity landing page built from scratch using pure HTML5 and CSS3 — no frameworks, no templates, no libraries. Developed as Task 3 of the CODSOFT Web Development Internship.
Perimeter is a fictional cybersecurity learning platform that offers hands-on ethical hacking labs, live attack simulations, and real-world security tasks for students.
