## ⚡ Special Additions

- 🤖 **3D CSS Robot** — animated floating bot in hero section
- 💬 **Quick Connect Widget** — floating helper button with resume & contact links  
- 🎊 **Confetti Burst** — fires when resume is downloaded
- ✨ **Scroll Reveal** — all sections animate as you scroll
- 🖱️ **Cursor Glow** — gradient follows mouse on desktop
- 👁️ **Live Viewer Badge** — shows today's viewer count
- 📲 **PWA** — installs as native app on Android, iOS, Windows
- 📴 **Offline Ready** — works without internet after first visit

---

## 🛠️ Tech Stack

```
HTML5          →  Structure & semantic markup
CSS3           →  Styling, animations, responsive layout
JavaScript     →  Interactivity, typing effect, confetti, scroll reveal
Web Manifest   →  PWA install configuration
Service Worker →  Offline caching support
GitHub Pages   →  Hosting & deployment
```
