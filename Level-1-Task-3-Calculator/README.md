# 📌 Project Overview

This project is a **Responsive Calculator Web Application** developed as **Level 1 – Task 3** of the **CODSOFT Web Development Internship**.

The calculator performs basic arithmetic operations with a clean and modern interface. It is fully responsive, supports keyboard input, and provides a smooth user experience across desktop and mobile devices.
